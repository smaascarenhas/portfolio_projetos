Projetos em Ciência de Dados

Desenvolvimento de um Painel Gerencial para Marketplace de entregas com Streamlit

Neste projeto, foram aplicados os princípios de programação em Python, manipulação, exploração e limpeza de dados, pensamento estratégico, lógica de negócios, além do uso de ferramentas de desenvolvimento web como Streamlit e GitHub, com o objetivo de desenvolver um painel gerencial com as principais métricas de uma empresa marketplace de restaurantes da Índia.

O resultado final do projeto foi um painel hospedado em um ambiente Cloud, acessível por meio de um link web. Este painel está disponível para acesso por qualquer dispositivo conectado à internet.

As ferramentas utilizadas foram:

  • Python
  • Jupyter Lab
  • Terminal
  • Streamlit
  • Streamlit Cloud
  • Github

Ensaio de Machine Learning

O projeto aborda os conceitos centrais de treinamento, ajuste de parâmetros e controle de overfitting e underfitting em diversos algoritmos de Machine Learning, nas áreas de classificação, regressão e agrupamento.

O resultado final consiste em tabelas com os melhores valores encontrados de performance dos algoritmos durante o ensaio de Machine Learning, com variações nos principais parâmetros que controlam o overfitting e underfitting.

As ferramentas utilizadas foram:

  • Python 3.11 e Scikit-learn
  • Algoritmos de classificação: KNN, Decision Tree, Random Forest e Logistic Regression
  • Métricas de performance: Accuracy, Precision, Recall, F1-Score, R2, MSE, RMSE, MAE, MAPE e Silhouette Score
  • Algoritmos de regressão: Linear Regression, Decision Tree Regressor, Random Forest Regressor, Polinomial Regression, Linear Regression Lasso, Linear Regression Ridge, Linear Regression Elastic Net, Polinomial Regression Lasso, Polinomial Regression Ridge e Polinomial Regression Elastic Net
  • Algoritmos de agrupamento: K-Means e Affinity Propagation
